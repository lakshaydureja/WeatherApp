Weather App



ABOUT

Welcome to the Weather App, your go-to source for up-to-date weather information from around the world! This app provides real-time data on the current weather conditions for any city you search, using an API to gather the latest information.
With our easy-to-use interface, you can quickly check the current temperature, humidity, wind speed, and other key weather indicators for your chosen location.

This WebApp is built with

  • HTML
  • CSS (No framework is used)
  • JavaScript
  • Node.js
  • Express.js
